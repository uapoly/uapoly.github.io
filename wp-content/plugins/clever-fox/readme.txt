== Installation ==

= Option 1. =

- Upload the entire `clever-fox` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Clever Fox" —> Click install.
- After plugin is downloaded, click active plugin
