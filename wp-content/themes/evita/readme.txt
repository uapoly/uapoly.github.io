== Description ==

Craft a standout resume, CV, or personal website with Evita, the sleek and customizable WordPress theme. Elevate your professional image with its modern design and user-friendly features, ensuring you make a memorable impression online. View the demo of Evita Pro https://www.nayrathemes.com/demo/pro/?theme=evita


== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
